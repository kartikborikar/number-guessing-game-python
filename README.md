# number-guessing-game-python
A simple and interactive CLI Number Guessing Game built using Python. Features basic loops, random number generation, and exception handling for invalid user inputs.

# Number Guessing Game 🎮

A classic and fun CLI-based game where the computer thinks of a random number between 1 and 100, and the player tries to guess it. 

## Features ✨
* **Random Generation:** Generates a new secret number every game using Python's `random` module.
* **Smart Feedback:** Guides the player by telling them if their guess is "Too high" or "Too low".
* **Attempt Tracker:** Counts and displays the total number of attempts taken to guess correctly.
* **Error Handling:** Gracefully handles invalid inputs (like text/letters) without crashing the game.
